Описание файлов:

 - 2_particles_15nm_Cone

   Структура из 22 слоев, толщина слоя 15нм, радиус слоев уменьшает при продвижении к вершине. Общая форма частицы - пирамида. Боковая сторона - ребристая. Добавлено вращение 5 град.

 - 2_particles_15nm_Prism

   Структура из 22 слоев, толщина слоя 15нм, радиус слоев постоянный. Общая форма частицы - призма. Боковая сторона - ребристая. Добавлено вращение 5 град. 
 
 - 2_particles_mix Prism+Cone_1_3_rotation

   Структура из 20 слоев, толщина слоя 15 нм. Первые 5 слоев имеют одинаковый диаметр. Начиная от 6 слоя и до вершины структуры радиус слоев уменьшается. Добавлено вращение 5 градусов. 

 - 2_particles_mix_Prism+Cone_1_1

   Структура из 20 слоев, толщина слоя 15 нм. Первые 10 слоев имеют одинаковый диаметр. Начиная с 11 слоя и до вершины структуры радиус слоев уменьшается. Добавлено вращение 5 градусов. 
 
 - 3Prism_tilt 30_int5+06

   3 структуры. Форма - Prism6. Из кремния. Повернуты по отношению к основанию на 30 градусов(ось y  для ВА). Развернуты относительно друг друга на 120 градусов (ось z для BA). Вся модель целиком вращается с шагом 5 градусов.

 - 3Prism_tilt 60_int5+06

   3 структуры. Форма - Prism6. Из кремния. Повернуты по отношению к основанию на 60 градусов(ось y  для ВА). Развернуты относительно друг друга на 120 градусов (ось z для BA). Вся модель целиком вращается с шагом 5 градусов.

 - 3047_SSDD_Cone6

   1 структура - шестигранная пирамида из кремния. Вращается с шагом 5 градусов.

 - 3047_SSDD_Cone6+Cap
   
   Core(Si)-shall(Au) шестигранная пирамида в основании и золотая частица на вершине. Вращение по z с шагом 5 градусов.

 - 3047_SSDD_Prism

   1 структура - шестигранная призма из кремния. Вращается с шагом 5 градусов.

 - 3047_SSDD_Prism6+Cap

   Core(Si)-shall(Au) шестигранная призма в основании и золотая частица на вершине. Вращение по z с шагом 5 градусов.

 - Cap1_int5+06
 
   Одна золотая  полусфера, поднятая на 300 нм (высоту основания НВ). Вращение. конечно, написано, но не имеет смысла.

 - Cap3_tilt 30_int5+06

   3 золотые полусферы, поднятые на высоту 379 нм (высота основания повернутого НВ). Повернутые относительно подложки (по оси y для ВА) на 30 градусов. Повернутые относительно друг друга (по оси z для ВА) на 120 градусов. Добавлено вращение по Z для всей конструкции с шагом в 5 градусов. 

 - Cap3_tilt 60_int5+06

   3 золотые полусферы, поднятые на высоту 379 нм (высота основания повернутого НВ). Повернутые относительно подложки (по оси y для ВА) на 60 градусов. Повернутые относительно друг друга (по оси z для ВА) на 120 градусов. Добавлено вращение по Z для всей конструкции с шагом в 5 градусов. 

 - For comparison of simulation and real data(3047_SSDD)

   Скрипт для составления 3-х изображений в одно.
